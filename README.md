# 🎮 Tic Tac Toe Game

A simple and fun Tic Tac Toe game built to play between two players on the same device.

## 🚀 Features
- Classic 3x3 grid gameplay
- Two-player mode (Player X and Player O)
- Detects win, draw, and restart game
- Simple and clean UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
